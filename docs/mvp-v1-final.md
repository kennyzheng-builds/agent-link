# Agent Link MVP v1.0 冻结方案

> 状态：已冻结，可直接开工  
> 更新时间：2026-03-07

---

## 一句话价值

让两个 OpenClaw Agent 像「老板的小弟」一样直接协作——人类只需拍板建联，Agent 自主沟通解决问题。

---

## 核心流程（三步）

```
1. 人工牵线 → 2. Agent 自主对话 → 3. 人类确认收尾
```

| 阶段 | 人类做什么 | Agent 做什么 |
|-----|-----------|-------------|
| **牵线** | 决定找谁，发送连接码/链接 | 生成/解析，建立加密通道 |
| **对话** | 随时查看进度、必要时介入 | 直接沟通、尝试解决、汇报关键节点 |
| **确认** | 验收结果、关闭会话 | 生成总结、同步给双方主人 |

---

## 连接机制

| 方式 | 示例 | 场景 |
|-----|------|------|
| **短链接（推荐）** | `https://link.openclaw.ai/j/A3F9K2` | 点击网页引导，或直接丢给 Agent |
| **连接码** | `A3F9K2` | 口述 |
| **兜底指令** | `/link join A3F9K2` | 备用 |

**链接流程**：
1. 用户点击链接 → 打开网页显示邀请信息
2. 用户复制指令 `/link join A3F9K2` 发给 Agent
3. 或直接把链接丢给 Agent，Agent 自动解析加入

## 身份与展示

### 身份体系
- **主身份**：Agent 本地 Ed25519 公钥（自托管，无外部依赖）
- **展示名**：
  1. 优先：用户自定义昵称
  2. 默认：邮箱前缀脱敏（如 `kennyz***`）
  3. 可选：完全匿名随机 ID

### 展示规则
| 场景 | 展示内容 |
|-----|---------|
| 会话列表 | 对方展示名 + 问题摘要 |
| 协作邀请 | "Kenny 的 Agent 邀请你协作" |
| 系统通知 | "已与 kennyz*** 建立连接" |

**隐私保护**：
- 不展示完整邮箱（保护域名隐私）
- 不展示真实 IP 或设备信息

## 技术架构

```
主人 A ◄──────► Agent a ◄──WebSocket──► Cloudflare Workers ◄──WebSocket──► Agent b ◄──────► 主人 B
                                              ↓
                                         仅转发密文
                                         不存储内容
```

| 组件 | 选型 | 成本 |
|-----|------|------|
| 信令服务 | Cloudflare Workers + Durable Objects | $0/月 |
| 传输层 | WebSocket (HTTPS) | 免费 |
| 加密 | X25519 + AES-256-GCM | 免费 |
| 身份 | Ed25519 自签名 | 免费 |

## 状态机（含异常）

```
[待创建] → [待加入] → [连接中] → [协作中] → [待确认] → [已关闭]
              ↓           ↓           ↓
           [过期]      [拒绝]      [超时]
              ↓           ↓           ↓
           自动回收    通知主人    自动关闭
```

| 状态 | 触发条件 | 处理 |
|-----|---------|------|
| 过期 | 连接码 10 分钟未使用 | 自动回收，通知创建者 |
| 拒绝 | 对方点击拒绝或忽略 | 通知创建者，会话关闭 |
| 超时 | 30 分钟无消息 | 提醒双方，60分钟后自动关闭 |
| 强制中止 | 任一方主人说"结束会话" | 立即关闭，生成总结 |

## 安全模型

### 介入优先级
```
主人指令 > Agent-Agent 对话
```
- 主人随时可打断
- Agent 立即响应
- 恢复后同步对端

### 信息同步
| 级别 | 内容 | 是否同步给对端主人 |
|-----|------|------------------|
| 关键决策 | "必须今天解决" / "预算不超过1000" | ✅ 必须 |
| 方案偏好 | "优先简单方案" | ✅ 同步 |
| 新线索 | "错误码是10003" | ✅ 同步 |
| 闲聊/情绪 | "好烦啊" / "加油" | ❌ 不同步 |

### 可见性
- **默认**：摘要视图（进度、关键决策、待办）
- **展开**：完整对话（a-b 之间的所有消息）

## MVP 功能清单（冻结）

### 必须实现（P0）
- [ ] Cloudflare Workers 信令服务（连接码生成/验证）
- [ ] WebSocket 消息转发（仅转密文）
- [ ] OpenClaw Skill 框架
- [ ] `/link` 指令集（create/join/status/close）
- [ ] 自然语言意图识别
- [ ] X25519 密钥交换
- [ ] AES-256-GCM 消息加密
- [ ] 本地会话存储
- [ ] 主窗口状态卡片

### 体验优化（P1）
- [ ] 短链接网页引导页
- [ ] 连接码二维码生成
- [ ] 完整对话视图
- [ ] 会话超时提醒

### 不做（MVP 外）
- [ ] 文件/图片传输
- [ ] 多人（>2）协作
- [ ] WebRTC P2P
- [ ] 经验分享社区
- [ ] Agent 能力标签
